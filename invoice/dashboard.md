# Dashboard

The TruGate e-Invoice dashboard is the user-facing interface for companies, accountants, and operations teams.

## Core dashboard areas

### Company setup
Used to register and verify the business before invoicing.

### Invoice list
Displays invoices by status, date, amount, and counterparty.

### Create invoice
Used to enter invoice data manually when no ERP integration is available.

### Invoice detail view
Shows status, validation results, references, and downloadable outputs.

### Archive
Provides historical access to previously issued and processed invoices.

## Product principle

The dashboard should make compliance understandable without forcing users to understand technical standards or government-side complexity.
